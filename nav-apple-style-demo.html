Deltas applied

  • ① Font stack -apple-system, "SF Pro Text" + letter-spacing: -0.022em cho display sizes
  • ② font-variant-numeric: tabular-nums cho count, sequence number, date — numbers align hàng dọc
  • ③ Active sprint: mx-2 rounded-lg bg-accent/12 text-accent-strong — bỏ border-left, dùng tint pill inset
  • ④ Project badge: rounded-full opacity-0.7 → 1 + active ring shadow 0 0 0 2px bg, 0 0 0 4px accent/35. Bỏ rust strip Discord-style
  • ⑤ Panel BG: --canvas-sunk: #f3f1ec (cooler hơn main #fafaf7) — tạo 2 tầng vibrancy. Hairline border rgba(0,0,0,0.06) thay --border

5 quick wins chi tiết

① FONT
SF Pro stack
macOS render SF Pro với optical sizing tự động (Text <20px, Display ≥20px). `letter-spacing: -0.022em` mô phỏng Display tracking cho headings.
② NUMS
Tabular numbers
`font-variant-numeric: tabular-nums` cho mọi count + date + sequence. Numbers align cột — quan trọng với task list dài.
③ PILL
Rounded tint active
Bỏ `border-left-2`. Active = `mx-2 rounded-lg bg-accent/12 text-accent-strong`. Inset margin cho cảm giác "selected within sidebar", không "kéo dài qua mép".
④ RING
Circular badge + ring
Bỏ Discord rust strip. Badge `rounded-full` inactive opacity 0.7, active opacity 1 + double box-shadow ring trên màu BG (Finder pattern).
⑤ TIERS
Sunk sidebar BG
Panel + rail dùng `--canvas-sunk: #f3f1ec` (darker). Main `--surface: #fff`. Hairline border 6% black thay solid `--border`. Tạo elevation rõ ràng.
