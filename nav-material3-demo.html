6 Material 3 deltas chi tiết

① FONT
Roboto Flex + Symbols
Variable font (opsz, wdth axes). Material Symbols Rounded với fill axis cho icons. Tracking nhẹ hơn SF Pro, geometric hơn.
② TONAL
3-tier surface
Derived từ rust #c04a1a thành tonal palette. Rail = container, panel = container-low, main = surface. Tạo "stacked layers" depth.
③ PILL
Filled rounded active
Active sprint = `rounded-full bg-secondary-container`. Height 40px (M3 NavigationDrawer spec). Bỏ border-left hoàn toàn.
④ STATE
State layer overlay
Hover = ::after pseudo với `bg-on-surface opacity 0.08`. Press = 0.12. Mọi interactive component có cùng tactile feedback.
⑤ FAB
Extended FAB
Primary action "New task" floating bottom-right. radius 16px, primary-container fill, 3-layer shadow (M3 elevation 3). Discoverable nhất.
⑥ RAIL
Icon + label rail
Width 80px (vs 60). Active pellet hình thuốc 56×32 với secondary-container fill. Label dưới (11px). M3 NavigationRail spec chuẩn.
